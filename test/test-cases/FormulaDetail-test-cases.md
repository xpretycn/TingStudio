## 一、页面概览

配方详情页以只读方式展示配方的完整信息，包含以下交互区域：
- **顶部 Header**：返回按钮、面包屑导航、配方名称/版本/状态标签、编辑和导出按钮
- **营养数据缺失警告**：缺失原料营养数据时显示
- **左侧栏**：配方概况卡片、关联业务员与需求、备注信息、制法、报价明细卡、变更记录时间线
- **右侧栏**：营养成分计算表格、含量比校验信息、营养成分表/技术处理依据
