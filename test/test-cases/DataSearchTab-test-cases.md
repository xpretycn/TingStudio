## 一、页面概览

DataSearchTab 是智能查询功能面板，支持自然语言查询原料和配方数据。核心流程：输入查询语句 → AI 分析意图生成 SQL → 执行查询返回结构化结果。页面包含：
1. **数据范围提示栏**：展示可检索的数据范围（配方、原料、业务员、配方销量）
2. **查询输入区**：textarea 输入框 + 智能查询按钮 + 导出CSV按钮
3. **快速示例标签**：预设的快捷查询标签，点击自动填入并发送
4. **加载状态**：查询中的 loading 提示
5. **错误提示区**：SQL 错误和 AI 错误的差异化提示
6. **SQL 查询卡片**：可折叠展示 AI 生成的 SQL 语句，带查询类型标签
7. **结果摘要**：显示结果条数和使用的模型
8. **结果表格**：结构化展示查询结果，支持 null 值显示
