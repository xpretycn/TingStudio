## 一、页面概览

周报详情页面展示单份周报的完整内容，包含：
- **头部区域**：返回按钮、面包屑导航、报告标题与状态标签、操作按钮（编辑/发布/导出PDF/导出Excel）
- **仪表盘区域**：统计卡片
- **配方完成情况**：每日配方趋势图、状态分布图、指标统计
- **销售数据统计**：每日销售趋势图、Top配方图、周对比图
- **未来规划**：可展开/折叠、可编辑的规划内容（下周计划/资源需求/预期目标/风险评估）
