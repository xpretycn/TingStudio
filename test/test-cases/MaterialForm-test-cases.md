## 一、页面概览

原料表单页用于新增和编辑原料，包含以下区域：
- **顶部Header**：面包屑导航、返回按钮、取消/提交审批/保存按钮、状态标签（草稿/审批中/已发布）、版本标签
- **版本化编辑提示横幅**：编辑已发布原料时显示，提示将创建新版本，展示引用配方列表
- **状态提示横幅**：新建时提示草稿状态，编辑草稿时提示可提交审批
- **基础信息区**：可折叠，含原料名称、类型、单位、编码、库存、单价、性状、口感、功效
- **营养成分区**：可折叠Excel导入面板、清空/展开/收起按钮、营养成分折叠面板（4个分组共26个营养素）、能量自动计算、数据来源/可信度/备注
- **表单校验**：原料名称必填且2-100字符、单位必填、库存必填、单价不能为负数

### 关键变更说明（相对 TC-MF-20260606-001）
1. 原料名称新增 `max:100` 校验规则
2. 单价新增负数校验规则（`validator: val >= 0`）
3. 新增版本化编辑模式横幅及引用配方列表的测试
4. 新增营养成分来源标签（NutritionSourceTag）的测试
5. 新增Excel导入子组件完整交互测试
