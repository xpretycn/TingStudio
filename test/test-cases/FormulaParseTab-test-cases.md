## 一、页面概览
FormulaParseTab 是智能填单功能面板，支持上传Excel/图片文件，通过AI解析配方信息。核心流程：上传文件 → 选择模板（可选）→ 开始解析 → 查看解析结果 → 编辑修正 → 提交配方。解析结果展示区包含配方基本信息（可编辑）、配方报价、原料列表（可编辑单价/用量）、提交按钮等。
