## 一、页面概览

仪表盘页面提供系统首页概览，采用 Bento Grid 布局，包含：
- **左栏**：审批卡片、近期动态（带分页）
- **右栏**：统计卡片（配方/原料/营收/报告）、快捷操作、精选配方、销量趋势图表（周/月/年切换）
