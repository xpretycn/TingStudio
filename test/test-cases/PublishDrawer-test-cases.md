## 一、页面概览

发布抽屉用于将快速配方发布为正式配方，从右侧滑出，宽度 420px。包含以下区域：
- **Header**：上传图标 + 标题"发布为正式配方" + 副标题说明
- **提示条**：蓝色信息提示"发布后快速配方状态将变为'已发布'，不可再编辑"
- **表单区**：业务员选择（必填，可搜索可清空）、配方描述（必填，maxlength=2000）、制备方法（选填，maxlength=5000）
- **底部 Footer**：取消按钮 + 确认发布按钮（校验通过后可点击）
