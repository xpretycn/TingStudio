## 一、页面概览

版本差异对比页用于对比同一配方的不同版本数据差异，支持最多 3 个版本并排对比。页面以卡片网格形式展示各版本，第一张卡片为基准版本，其余卡片与之对比。支持含量模式和报价模式切换，差异通过颜色和样式高亮标记。

**页面布局**：
- **顶部导航栏**：返回按钮、面包屑导航、配方名称与版本数标签、对比计数、模式切换按钮、重置对比按钮
- **主体区域**：版本对比卡片网格 + 添加版本占位卡片
- **空状态**：无选中版本时的引导提示

**核心业务逻辑**：
- 对比列表从 localStorage（`compare_versions`）读取，页面加载时恢复
- 第一个卡片固定为基准版本，其他版本与之对比
- 含量模式：原料按名称对齐，差异标记为 `diff-added`（绿色）/ `diff-changed`（琥珀色）/ `diff-missing`（红色虚线框）
- 报价模式：同样按名称对齐，含调价标记（`pi-adjust-badge`）和小计计算
- 费用与利润区：包材费用、其他费用、利润率与基准对比，标记上涨/下降/变化
- 版本摘要根据状态显示不同文案
