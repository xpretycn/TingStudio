## 一、页面概览

ParseHistoryTab 是解析历史功能面板，展示所有 AI 解析记录。页面分为四个区域：
1. **数据概览卡片**：4张统计卡片（总数、存储状态、待处理、已关联），可点击筛选
2. **工具栏**：批量操作栏（选中时显示）、状态筛选下拉、时间范围选择器、搜索框、刷新/重置按钮
3. **历史列表**：卡片式展示，每条记录含复选框、类型标签、状态标识、文件信息、删除按钮
4. **分页控件**：自定义分页，显示当前范围和总条数
5. **详情抽屉**：t-drawer 展示解析详情，含文件信息、解析结果、操作按钮
6. **降级提示**：DegradationBanner 组件，功能降级时显示
