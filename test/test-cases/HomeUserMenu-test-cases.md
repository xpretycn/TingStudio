## 一、页面概览

用户菜单位于 Home 页面顶部 Header 右侧，通过 hover 用户头像区域触发 t-popup 下拉菜单。菜单包含：账号设置、切换外观（子菜单含3种主题模式）、切换品牌色（子菜单含4种品牌色）、切换账号、退出登录。子菜单通过 hover 触发，向右展开。
