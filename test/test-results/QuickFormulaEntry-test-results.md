## 重要说明

QuickFormulaEntry 组件（居中卡片布局，包含图标区、输入框、开始编辑按钮、草稿区）在当前代码版本中**已被重构移除**。当前快速配方页面（`/formulas/quick`）使用的是侧边栏列表 + 编辑区域的布局（QuickFormulaPanel + QuickFormulaSidebar），不再包含 QuickFormulaEntry 组件。

源文档中描述的 `.quick-formula-entry`、`.entry-card`、`.entry-icon`、`.entry-title`、`.entry-desc`、`.entry-form`、`.draft-section` 等 CSS 类在当前页面中均不存在。
