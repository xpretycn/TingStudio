## 二、功能需求

### 2.1 功能一：日期选择优化

**问题描述**：当前周报生成需手动选择每周日期，操作流程过于复杂。

**优化方案**：重构为按月份选择周次的模式，利用ISO周次规则实现简化操作。

**交互设计**：

| 组件 | 说明 |
|------|------|
| 月份下拉框 | 选择目标月份（当前月份默认选中） |
| 周次下拉框 | 根据所选月份显示当月包含的ISO周数（1-5周） |
| 日期预览 | 显示所选周次的实际日期范围（周一~周日） |

**ISO周次规则**：
- 每周从**周一**开始，到**周日**结束
- 第1周：包含该年1月4日的那个周
- 周次计算遵循 ISO 8601 标准

**周次下拉框选项示例**（以2026年5月为例）：
```
2026年5月
├── 第1周 (05-04 ~ 05-10)
├── 第2周 (05-11 ~ 05-17)
├── 第3周 (05-18 ~ 05-24)
├── 第4周 (05-25 ~ 05-31)
```

### 2.2 功能二：权限管理增强

**需求**：为报告中心添加精细化权限控制机制。

**权限矩阵**：

| 角色 | 查看范围 | 创建 | 编辑 | 删除 | 发布 | 导出 |
|------|---------|------|------|------|------|------|
| **admin** | 所有报告 | ✅ | ✅ | ✅ | ✅ | ✅ |
| **配方师** | 自己创建的报告 + 已发布报告 | ✅ | 仅自己的报告 | 仅自己的报告 | ❌ | 仅自己的报告 |

**约束条件**：
- **同一角色、同一周期只能生成一次报告**
- 示例：配方师张三在2026年第22周已生成周报，则同一周期不可再次生成

**数据隔离实现**：

| 层级 | 实现方式 |
|------|---------|
| 后端API | `reports` 表查询添加 `created_by` 条件过滤 |
| 前端界面 | 根据用户角色动态显示操作按钮 |

---

### 2.3 功能三：查看与导出功能优化

**现有功能**：
- `GET /reports/:id/export/pdf` - 导出PDF
- `GET /reports/:id/export/excel` - 导出Excel

**优化方向**：

| 功能点 | 当前状态 | 优化目标 |
|--------|---------|---------|
| 单个导出 | 跳转到报告详情页 | 提供导出格式选择（PDF/Excel） |
| 批量导出 | 仅提示"已选择N个报告" | 实现批量Excel导出功能 |
| 导出下载 | Blob下载，无loading状态 | 添加导出中loading状态和成功提示 |

**导出格式支持**：

| 格式 | 技术实现 | 文件名格式 |
|------|---------|-----------|
| PDF | pdfkit（后端生成） | `周报_2026-W22_20260525.pdf` |
| Excel | xlsx（SheetJS，后端生成） | `周报_2026-W22_20260525.xlsx` |

## 三、用户操作流程

### 3.1 生成周报流程

```
┌─────────────────────────────────────────────────────────────────┐
│                      生成周报交互流程                              │
├─────────────────────────────────────────────────────────────────┤
│  1. 用户点击「生成周报」按钮                                      │
│         ↓                                                        │
│  2. 弹出抽屉组件，包含：                                          │
│     ├── 月份下拉框（默认当前月份）                                 │
│     ├── 周次下拉框（显示当月ISO周次）                              │
│     └── 日期范围预览（显示周一~周日）                              │
│         ↓                                                        │
│  3. 用户选择月份 → 系统自动计算该月包含的ISO周次                    │
│     用户选择周次 → 系统显示该周实际日期范围                          │
│         ↓                                                        │
│  4. 系统校验：检查同一用户同一周期是否已生成报告                     │
│     ├── 未生成 → 启用「生成」按钮                                  │
│     └── 已生成 → 禁用按钮，提示"该周报已存在"                        │
│         ↓                                                        │
│  5. 用户点击「生成」→ 显示loading状态                              │
│         ↓                                                        │
│  6. 成功后 → 关闭抽屉，刷新列表，Toast提示"周报生成成功"             │
│     失败后 → 错误提示，可重试                                       │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 查看报告流程

```
┌─────────────────────────────────────────────────────────────────┐
│                      查看报告交互流程                              │
├─────────────────────────────────────────────────────────────────┤
│  1. 列表页展示报告卡片，按月份分组                                  │
│         ↓                                                        │
│  2. 配方师登录：仅显示自己创建的报告 + 已发布报告                    │
│     admin登录：显示所有报告                                        │
│         ↓                                                        │
│  3. 点击报告卡片 → 跳转至报告详情页                                │
│     ├── 查看报告数据                                               │
│     ├── 导出报告（PDF/Excel）                                      │
│     └── 编辑/删除（仅报告创建者或admin可见）                        │
└─────────────────────────────────────────────────────────────────┘
```

### 3.3 导出报告流程

```
┌─────────────────────────────────────────────────────────────────┐
│                      导出报告交互流程                              │
├─────────────────────────────────────────────────────────────────┤
│  场景A：单个导出                                                  │
│  1. 悬停报告卡片 → 显示操作按钮（导出图标）                         │
│  2. 点击导出图标 → 显示导出格式选择弹窗（PDF / Excel）              │
│  3. 选择格式 → 显示loading → 下载文件                              │
│                                                                  │
│  场景B：批量导出                                                  │
│  1. 点击报告卡片复选框 → 选中状态                                  │
│  2. 选中多个报告 → 显示批量操作栏                                   │
│  3. 点击「批量导出」→ 下载打包的Excel文件（多Sheet）               │
└─────────────────────────────────────────────────────────────────┘
```

## 四、数据校验规则

| 规则ID | 规则描述 | 错误提示 |
|--------|---------|---------|
| R01 | 周报周期唯一性校验 | "该周期的周报已存在，请勿重复生成" |
| R02 | 月报周期唯一性校验 | "该月份的月报已存在，请勿重复生成" |
| R03 | 日期范围有效性校验 | "请选择有效的日期范围" |
| R04 | 权限校验 | "无权限查看此报告" / "无权限删除此报告" |
| R05 | 报告存在性校验 | "报告不存在或已被删除" |
