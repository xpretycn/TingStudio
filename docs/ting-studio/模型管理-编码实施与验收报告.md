## 5. API 端点汇总

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/ai/models-manage | 获取模型列表+统计 |
| POST | /api/ai/models-manage | 新增模型 |
| PUT | /api/ai/models-manage/:id | 更新模型 |
| DELETE | /api/ai/models-manage/:id | 删除模型 |
| POST | /api/ai/models-manage/:id/test | 测试连通性 |
| GET | /api/ai/models-manage/:id/versions | 获取版本列表 |
| PUT | /api/ai/models-manage/:id/fallback | 设置备用模型 |
| GET | /api/ai/usage | 用量统计 |
| GET | /api/ai/usage/logs | 调用日志 |
| GET | /api/ai/alerts/configs | 预警配置 |
| PUT | /api/ai/alerts/configs/:id | 更新预警配置 |
| GET | /api/ai/alerts/records | 预警记录 |
| GET | /api/ai/health | 健康状态 |
| GET | /api/ai/health/:provider/history | 健康历史 |
