# 模型管理功能 — 数据库设计文档

> 版本：v1.0 | 日期：2026-05-05 | 数据库：SQLite (better-sqlite3)

---

## 1. 表总览

| 表名 | 说明 | 关联 |
|------|------|------|
| ai_models | AI 模型配置 | 主表 |
| ai_usage_logs | 调用日志 | → ai_models.provider |
| ai_alert_configs | 预警配置 | → ai_models.id |
| ai_alert_records | 预警记录 | → ai_models.provider |
| ai_health_records | 健康检测记录 | → ai_models.provider |
| ai_fallback_configs | 备用模型配置 | → ai_models.id |

---

## 2. 表结构详情

### 2.1 ai_models — AI 模型配置

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | TEXT | PK | 主键，格式 `model_{provider}` |
| provider | TEXT | NOT NULL, UNIQUE | 模型提供商标识 |
| name | TEXT | NOT NULL | 显示名称 |
| base_url | TEXT | NOT NULL | API 基础地址 |
| api_key | TEXT | DEFAULT '' | API 密钥（加密存储） |
| model | TEXT | NOT NULL | 默认模型版本 |
| vision_model | TEXT | DEFAULT '' | 视觉模型版本 |
| vision_max_tokens | INTEGER | DEFAULT NULL | 视觉模型最大 Token |
| description | TEXT | DEFAULT '' | 模型描述 |
| supports_vision | INTEGER | NOT NULL DEFAULT 0 | 是否支持视觉（0/1） |
| health_status | TEXT | NOT NULL DEFAULT 'unknown' | 健康状态：healthy/degraded/unhealthy/unknown |
| last_health_check | TEXT | DEFAULT NULL | 上次检测时间 |
| last_health_latency | INTEGER | DEFAULT NULL | 上次检测延迟(ms) |
| health_check_interval_days | INTEGER | NOT NULL DEFAULT 1 | 健康检测间隔（天） |
| sort_order | INTEGER | NOT NULL DEFAULT 0 | 排序序号 |
| created_at | TEXT | NOT NULL DEFAULT datetime('now') | 创建时间 |
| updated_at | TEXT | NOT NULL DEFAULT datetime('now') | 更新时间 |

**索引：**
- `idx_ai_models_provider` ON (provider)
- `idx_ai_models_health` ON (health_status)

---

### 2.2 ai_usage_logs — 调用日志

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | TEXT | PK | 主键，UUID |
| provider | TEXT | NOT NULL | 模型提供商标识 |
| model_name | TEXT | NOT NULL | 模型显示名称 |
| model | TEXT | NOT NULL | 实际使用的模型版本 |
| call_type | TEXT | NOT NULL | 调用类型：parse_formula/parse_nutrition/natural_search/health_check |
| prompt_tokens | INTEGER | NOT NULL DEFAULT 0 | 输入 Token 数 |
| completion_tokens | INTEGER | NOT NULL DEFAULT 0 | 输出 Token 数 |
| total_tokens | INTEGER | NOT NULL DEFAULT 0 | 总 Token 数 |
| latency_ms | INTEGER | DEFAULT NULL | 响应延迟(ms) |
| status | TEXT | NOT NULL DEFAULT 'success' | 状态：success/error/fallback |
| error_message | TEXT | DEFAULT NULL | 错误信息 |
| request_summary | TEXT | DEFAULT NULL | 请求摘要 |
| user_id | TEXT | DEFAULT NULL | 调用用户 ID |
| fallback_from | TEXT | DEFAULT NULL | 原始 provider（自动切换时） |
| created_at | TEXT | NOT NULL DEFAULT datetime('now') | 创建时间 |

**索引：**
- `idx_usage_logs_provider` ON (provider)
- `idx_usage_logs_call_type` ON (call_type)
- `idx_usage_logs_status` ON (status)
- `idx_usage_logs_created_at` ON (created_at)

---

### 2.3 ai_alert_configs — 预警配置

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | TEXT | PK | 主键，格式 `alert_{provider}` |
| model_id | TEXT | NOT NULL | 关联 ai_models.id |
| provider | TEXT | NOT NULL | 模型提供商标识 |
| daily_call_limit | INTEGER | NOT NULL DEFAULT 500 | 日调用上限 |
| monthly_token_limit | INTEGER | NOT NULL DEFAULT 5000000 | 月 Token 上限 |
| warning_threshold | INTEGER | NOT NULL DEFAULT 80 | 预警阈值(%) |
| critical_threshold | INTEGER | NOT NULL DEFAULT 95 | 严重阈值(%) |
| enabled | INTEGER | NOT NULL DEFAULT 1 | 是否启用（0/1） |
| created_at | TEXT | NOT NULL DEFAULT datetime('now') | 创建时间 |
| updated_at | TEXT | NOT NULL DEFAULT datetime('now') | 更新时间 |

---

### 2.4 ai_alert_records — 预警记录

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | TEXT | PK | 主键，UUID |
| provider | TEXT | NOT NULL | 模型提供商标识 |
| model_name | TEXT | NOT NULL | 模型显示名称 |
| alert_type | TEXT | NOT NULL | 预警类型：daily_call/monthly_token |
| level | TEXT | NOT NULL | 级别：warning/critical |
| threshold | INTEGER | NOT NULL | 触发阈值(%) |
| current_value | INTEGER | NOT NULL | 当前值 |
| limit_value | INTEGER | NOT NULL | 限制值 |
| message | TEXT | NOT NULL | 预警消息 |
| is_read | INTEGER | NOT NULL DEFAULT 0 | 是否已读（0/1） |
| created_at | TEXT | NOT NULL DEFAULT datetime('now') | 创建时间 |

**索引：**
- `idx_alert_records_provider` ON (provider)
- `idx_alert_records_level` ON (level)
- `idx_alert_records_is_read` ON (is_read)

---

### 2.5 ai_health_records — 健康检测记录

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | TEXT | PK | 主键，UUID |
| provider | TEXT | NOT NULL | 模型提供商标识 |
| status | TEXT | NOT NULL | 检测结果：healthy/degraded/unhealthy |
| latency_ms | INTEGER | DEFAULT NULL | 响应延迟(ms) |
| error_message | TEXT | DEFAULT NULL | 错误信息 |
| checked_at | TEXT | NOT NULL DEFAULT datetime('now') | 检测时间 |

**索引：**
- `idx_health_records_provider` ON (provider)
- `idx_health_records_checked_at` ON (checked_at)

---

### 2.6 ai_fallback_configs — 备用模型配置

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | TEXT | PK | 主键，UUID |
| model_id | TEXT | NOT NULL | 关联 ai_models.id |
| fallback_provider | TEXT | NOT NULL | 备用模型 provider |
| fallback_priority | INTEGER | NOT NULL DEFAULT 1 | 优先级（1=最高） |
| enabled | INTEGER | NOT NULL DEFAULT 1 | 是否启用（0/1） |
| created_at | TEXT | NOT NULL DEFAULT datetime('now') | 创建时间 |
| updated_at | TEXT | NOT NULL DEFAULT datetime('now') | 更新时间 |

## 3. 初始数据

应用首次启动时，`ensureInitialAiModels()` 自动从环境变量迁移以下模型：

| provider | name | model | vision_model |
|----------|------|-------|-------------|
| dashscope | 通义千问 | qwen-plus | qwen-vl-plus |
| zhipu | 智谱清言 | glm-4-flash | glm-4v-flash |
| deepseek | DeepSeek | deepseek-chat | - |

每个模型同时创建默认预警配置：
- 日调用上限：500
- 月 Token 上限：5,000,000
- 预警阈值：80%
- 严重阈值：95%

```
    ├── (N) ai_alert_records

    │
    ├── (N) ai_health_records
    │              via provider
    │
    └── (1:N) ai_fallback_configs

```
