# 悬浮球Agent增强 — 手动验收方案

> 版本: 1.0.0 | 日期: 2026-05-14 | 范围: 悬浮球AI助手FloatAgent增强功能

---

## 验收目标

| # | 目标 | 说明 |
|---|------|------|
| 1 | 双模路由正确性 | 输入文本被正确分类为 `fill`（表单填字段）或 `agent`（工具调用对话） |
| 2 | 7类意图识别准确性 | compare/substitute/quotation/generate/calculate/consult/fill 关键词匹配无误 |
| 3 | SSE 流式对话可用 | 悬浮球中 Agent 对话走 SSE 协议，流式输出正常 |
| 4 | 3个新工具调用成功 | `compare_formulas` / `suggest_material_substitute` / `generate_quotation` |
| 5 | 3个结果卡片渲染正确 | CompareCard / QuotationCard / SubstituteCard 可视化渲染无异常 |
| 6 | UI 增强功能完整 | 状态灯/角标/快捷命令/动态标题/指令模板栏均可用 |
| 7 | 悬浮助手配置可用 | 管理员可配置模型/外观/行为策略，配置生效 |
| 8 | 错误处理与降级 | 网络异常/模型超时/工具失败等场景有合理降级 |

---
