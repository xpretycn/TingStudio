# 业务员表单页测试用例

## 页面信息
| 项目 | 值 |
|------|------|
| 页面名称 | 新增业务员 / 编辑业务员 |
| 路由 | /salesmen/new, /salesmen/:id/edit |
| 文件 | frontend/src/views/salesmen/SalesmanForm.vue |

## 交互元素清单

| 编号 | 元素名称 | 类型 | 选择器 | 说明 |
|------|---------|------|--------|------|
| E01 | 返回按钮 | button | `.header-back-btn` | 返回列表页 |
| E02 | 面包屑链接 | a | `.breadcrumb-link` | 点击"业务员管理"返回列表 |
| E03 | 取消按钮 | button | `.header-action-btn.secondary` | 取消编辑，返回列表 |
| E04 | 提交按钮 | button | `.header-action-btn`（非 secondary） | 新增"创建"/编辑"保存" |
| E05 | 姓名输入框 | t-input | `input[data-field="name"]` | 必填，2-20字符 |
| E06 | 工号输入框 | t-input | `input[data-field="code"]` | 必填，字母数字下划线横线 |
| E07 | 部门输入框 | t-input | `input[data-field="department"]` | 选填，最多50字符 |
| E08 | 电话输入框 | t-input | `input[data-field="phone"]` | 选填，11位手机号 |
| E09 | 邮箱输入框 | t-input | `input[data-field="email"]` | 选填，邮箱格式 |
| E10 | 头像上传区域 | div | `.upload-zone` | 点击/拖拽上传，JPG/PNG，最大2MB |
| E11 | 头像文件输入 | input[file] | `input[type="file"]` | 隐藏文件选择器 |
| E12 | 头像预览区域 | div | `.preview-area` | 预览+更换/删除按钮 |
| E13 | 更换头像按钮 | button | `.preview-btn--primary` | 重新选择头像 |
| E14 | 删除头像按钮 | button | `.preview-btn--danger` | 删除已上传头像 |
