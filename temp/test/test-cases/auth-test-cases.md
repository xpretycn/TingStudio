## 一、页面概览

### 登录页（Login.vue）
左右分栏布局。左侧为品牌展示区（渐变背景 + AnimatedCharacters 角色动画），右侧为毛玻璃表单卡片。表单包含用户名输入框、密码输入框（含可见性切换）、登录按钮、注册链接。

### 注册页（Register.vue）
与登录页一致的左右分栏布局。左侧同样为品牌展示区 + 角色动画，右侧表单卡片包含用户名、密码、确认密码三个输入框（密码类均含可见性切换）、注册按钮、登录链接。

### 共性特征
- 输入框 focus 时触发 `isTyping`，驱动左侧角色动画
- 密码输入触发 `hasSecret`，密码可见时触发 `secretVisible`，角色做出躲藏/偷看动画
- 表单校验失败时 TDesign 自动标红字段
- 登录/注册成功后跳转首页 `/`
- 登录/注册失败弹出 MessagePlugin.error 提示
