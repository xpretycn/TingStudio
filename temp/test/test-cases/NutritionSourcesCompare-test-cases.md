## 一、页面概览

营养数据来源对比页面用于对比同一原料的多个营养数据来源（手工录入、天行API、种子库、AI估算、Excel导入等），帮助用户识别各来源间的偏差，选择最可信的主用值。页面采用左右分栏布局，左侧为来源列表面板，右侧为对比视图（概览/数值对比/偏差分析/历史四个 Tab），顶部有导航栏和导出功能。

**页面布局**：
- **顶部导航栏**：返回按钮、面包屑导航、原料名称+来源数+平均偏差标签、变更历史按钮、导出报告下拉
- **左侧面板**：来源列表面板（SourceListPanel），含类型筛选、来源项列表
- **右侧区域**：雷达图 + 4 个 Tab 视图（概览/数值对比/偏差分析/历史）
- **底部浮动栏**：批量操作栏（SourceBatchActionBar），选中来源时显示
- **对话框**：批量设为主用对话框（SourceSetAuthoritativeDialog）、变更历史对话框（SourceSnapshotDialog）
- **快捷键提示**：底部浮动快捷键提示条

**核心业务逻辑**：
- 页面加载时并行请求 4 个接口：来源列表、评分来源、来源对比、推荐来源
- 通过多数投票算法（`computeMajorityAuthoritative`）从 `fieldSources` 推断当前主用来源 ID
- 支持 4 种主用值选取策略：综合评分（best-deviation）、最高可信度（highest-confidence）、最新时间（newest）、手动指定（manual）
- 偏差严重度分级：<1% success、<3% info、<5% warning、>=5% danger
- 雷达图展示最多 6 个来源的 6 大营养指标（能量/蛋白质/脂肪/碳水/钠/纤维）
- 导出支持 Excel 和 PDF 两种格式
- 快捷键：1-4 切换视图、? 切换快捷键提示、Escape 关闭弹窗/清空选择

**权限控制**：
- `canEdit`（设为主用值）：仅 admin
- `canArchive`（批量归档）：admin + formulist
