## 一、页面概览

模板管理器是一个 t-dialog 弹窗组件，分为两个区域：
- **保存区**：将当前配方数据保存为模板，仅在有原料时显示（canSave）
- **已保存区**：浏览、搜索、加载和删除已有模板

核心业务流程：打开弹窗 → 自动加载模板列表 → 保存/搜索/加载/删除模板 → 关闭弹窗
