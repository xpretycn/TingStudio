## 一、页面概览

原料详情页展示原料的完整信息，包含以下区域：
- **顶部Header**：返回按钮、面包屑导航（原料管理 > 原料详情）、原料名称/类型标签(药材/辅料)/版本标签/状态标签(草稿/审批中/已发布)、编辑原料按钮(权限控制)、导出原料按钮
- **左侧栏**：
  - 原料概况卡片：原料编码、名称、类型、单位、库存数量(带状态色)、营养状态(带状态色)、单价(带状态色)、性状标签、口感标签、功效标签、创建时间、更新时间
  - 变更记录时间线：当前版本/初始录入、时间、备注
- **右侧栏**：
  - 营养成分区：标题(原料名称+营养成分)、每100g含量分析、数据源标签(NutritionSourceTag)、可信度标签(高/中/低)、打开对比页按钮、营养数据网格(含进度条和百分比)、空状态
  - 能量计算区：NRV占比(含进度条和超标标识)、总能量卡片(含三大营养素占比堆叠条)、换算公式卡片(蛋白质x17、脂肪x37、碳水x17)、空状态
- **导出抽屉**：导出格式单选组(Excel/PDF)、导出模板下拉框(可筛选)、取消/开始导出按钮

### 关键业务逻辑说明
1. **canEdit权限**：admin可编辑所有原料；formulist仅可编辑自己创建的原料（通过 `material.createdBy === user.id` 判断）
2. **NRV计算**：已修复钠的 /1000 错误逻辑，统一使用 g->mg 转换（当数据单位是g但NRV参考值单位是mg时，乘以1000）
3. **能量计算**：蛋白质x17 + 脂肪x37 + 碳水x17，结果取整
4. **营养进度条**：基于预设maxValues计算百分比，上限100%
5. **返回导航**：携带当前路由query参数返回列表页
