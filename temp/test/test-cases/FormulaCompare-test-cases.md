## 一、页面概览

配方对比页支持最多3个配方的多维对比，包含以下交互区域：
- **顶部 Header**：返回按钮、面包屑导航、对比数量显示、模式切换按钮、重置对比按钮
- **空状态**：无选中配方时显示引导
- **对比卡片网格**：每个配方一张卡片，显示原料构成/报价、费用与利润、配方摘要
- **添加对比占位卡片**：可从可选列表中添加新配方
- **对比模式切换**：含量对比 / 报价对比
