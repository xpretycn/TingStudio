## 一、页面概览

配方列表页是配方管理的核心入口，包含以下交互区域：
- **顶部数据看板**：FormulaDashboardCards 组件，展示统计卡片
- **工具栏**：搜索框、快速录入、批量录入销量、创建新配方、刷新、筛选按钮
- **批量操作栏**：选中行后显示，包含批量删除、批量归档、导出、配方对比
- **配方数据表格**：展示配方列表，支持行展开查看版本记录、行选择、行点击跳转详情
- **操作菜单**：每行的更多操作（发布/提交审批、版本管理、编辑、录入销量、删除）
- **分页控件**：上一页、页码、下一页
- **右侧面板**：最近动态、待办事项
- **导出抽屉**：导出配方数据
- **批量录入销量抽屉**：批量录入销量数据
- **销量录入对话框**：单条配方录入销量
