## 一、页面概览

原料版本对比页面用于对比同一原料的不同版本数据差异。页面以卡片网格形式展示各版本，每张卡片显示基本信息和营养成分的对比结果，差异字段通过颜色和徽标高亮。支持最多 3 个版本同时对比，以第一个选中的版本为基准进行差异计算。

**页面布局**：
- **顶部导航栏**：返回按钮、面包屑导航、原料名称、版本数标签、重置对比按钮
- **主体区域**：版本对比卡片网格 + 添加版本占位卡片
- **空状态**：无选中版本时的引导提示

**核心业务逻辑**：
- 从 URL query 参数 `v1`/`v2`/`v3` 读取预选版本 ID
- 若无 query 参数，默认选择最新 2-3 个版本自动对比
- 对比以第一个版本为基准，调用 `materialApi.compareVersions` 逐对比较
- 差异合并算法 `mergeDiffItems` 将多对比较结果合并为统一视图
- 差异类型：`increase`（增长）、`decrease`（减少）、`unchanged`（不变）、`new`（新增）、`deleted`（删除）
