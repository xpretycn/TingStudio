## 一、页面概览
MaterialImportTab 是智能导入功能面板，支持上传Excel/图片文件，通过AI解析原料营养数据。核心流程：上传文件 → 选择模板（可选）→ 开始解析 → 查看解析结果（原料营养数据表格）→ 编辑修正 → 批量导入。解析结果以表格形式展示，支持行内编辑、置信度显示、重复检测、数据校验等功能。
