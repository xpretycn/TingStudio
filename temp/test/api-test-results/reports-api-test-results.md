## 失败用例详情

### D05-P01 — 创建目标返回 200 而非 201
- **严重程度**: Low
- **预期状态码**: 201
- **实际状态码**: 200
- **预期响应**: `{success:true, data:{id,...}, message:"创建成功"}` 状态码 201
- **实际响应**: `{success:true, data:{id,...}}` 状态码 200
- **修复建议**: 创建目标成功时应返回 HTTP 201 状态码，当前返回 200。修改 reportController.createTarget 中 res.status(201) 替换 res.json()。

### D11-P01 — 生成周报返回 200 而非 201
- **严重程度**: Low
- **预期状态码**: 201
- **实际状态码**: 200
- **预期响应**: `{success:true, data:{id,...}, message:"报告生成成功"}` 状态码 201
- **实际响应**: `{success:true, data:{id,...}}` 状态码 200
- **修复建议**: 生成报告成功时应返回 HTTP 201 状态码。修改 reportController.generateReport 中 res.status(201)。
