## 五、风险与注意事项

1. **配方数据修复**：当前 180 条配方的 `materials_json` 引用了已删除的 MAT001-MAT030，建议重建数据库而非逐条修复
2. **营养数据完整性**：当前 52 条营养数据仅包含能量、蛋白质、脂肪、碳水化合物、钠 5 个字段，PRD 要求 25+ 个字段（维生素、矿物质等），后续需补充
3. **前端 LocalStorage 残留**：早期版本使用 LocalStorage 存储数据（`initData.ts`、`storage.ts`），需确认是否已完全迁移至后端 API
4. **EdgeOne Pages 部署**：当前前端已部署，但后端 API 服务需单独部署才能实现完整的前后端联调
