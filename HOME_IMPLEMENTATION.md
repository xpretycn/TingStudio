# 主页实现说明

## 概述
成功设计并实现了一个与登录页风格一致的主页，采用左右分栏布局，左侧为导航功能区，右侧为内容展示区。

## 实现内容

### 1. 新建文件
- **`/src/views/Home.vue`**: 主页布局组件
- **`/src/views/Tools.vue`**: 工具箱页面

### 2. 修改文件
- **`/src/router/index.ts`**: 更新路由配置，将主页路由指向新的Home组件

### 3. 按钮样式统一
更新了以下页面的按钮样式，确保与主页风格一致：
- `/src/views/formulas/FormulaList.vue`
- `/src/views/materials/MaterialList.vue`
- `/src/views/customers/CustomerList.vue`

## 功能特性

### 左侧导航功能区
1. **Logo区域**
   - 可爱的小猫SVG图标（带弹跳动画）
   - 应用名称"TingStudio"
   - 与登录页风格完全一致

2. **个人信息**
   - 用户头像（SVG图标）
   - 用户名和角色标签
   - 圆角卡片设计，带毛玻璃效果

3. **日期和天气**
   - 当前日期（YYYY-MM-DD格式）
   - 星期显示
   - 天气状况（晴天、多云等）
   - 温度显示

4. **今日祝福语**
   - 随机显示温馨祝福语
   - 心形图标动画效果

5. **可折叠导航栏**
   - 点击标题可展开/收起
   - 5个导航项：
     * 最近配方 (🕐)
     * 配方管理 (📝)
     * 原材料管理 (🧪)
     * 客户管理 (👥)
     * 工具箱 (🛠️)
   - 当前激活项高亮显示
   - 悬停效果和箭头动画

6. **退出登录按钮**
   - 统一的粉色主题风格
   - 点击动画效果

### 右侧内容展示区
1. **内容头部**
   - 页面标题
   - 新增按钮（带图标）
   - 刷新按钮（带图标）

2. **动态内容区**
   - 通过`<router-view>`动态加载内容
   - 淡入淡出过渡动画
   - 支持所有子页面的增删改查功能

### 工具箱页面
提供6个常用工具入口：
- 🧮 配方计算器
- 📊 数据统计
- 🔍 配方搜索
- 📋 模板管理
- 💾 数据备份
- ⚙️ 系统设置

## 设计风格统一

### 颜色方案
与登录页完全一致：
- 主色调：粉色渐变 (#FF8FAB → #FF6B8A)
- 背景色：#FFF9F5（奶油白）
- 文字色：#5D4E60（主文字）、#9B8FA0（次要文字）
- 辅助色：#FFE8D6（桃色）、#F0E6FF（淡紫色）

### 按钮设计
所有按钮统一样式：
- **主按钮**：粉色渐变背景，带图标，点击动画
- **默认按钮**：白色背景，粉色边框
- **危险按钮**：红色渐变背景
- **文字按钮**：悬停时粉色背景
- 所有按钮带统一的过渡动画和阴影效果

### 动画效果
- Logo小猫弹跳动画（4s循环）
- 心形跳动动画
- 导航项悬停效果
- 按钮点击反馈（缩放+阴影变化）
- 页面切换淡入淡出

## 响应式设计
- 平板端（≤1024px）：缩小侧边栏宽度
- 移动端（≤768px）：转换为垂直布局，侧边栏变为顶部区域
- 所有内容页面都已适配移动端

## 增删改查功能

### 已有功能
所有内容页面（配方管理、原材料管理、客户管理）都包含完整的CRUD功能：

1. **配方管理** (`/src/views/formulas/`)
   - 列表展示（支持搜索、筛选）
   - 新增配方
   - 编辑配方
   - 删除配方（带确认）
   - 查看配方详情（可展开查看原料清单）

2. **原材料管理** (`/src/views/materials/`)
   - 列表展示（支持搜索）
   - 新增原料
   - 编辑原料
   - 删除原料（检查是否被配方使用）

3. **客户管理** (`/src/views/customers/`)
   - 列表展示（支持搜索）
   - 新增客户
   - 编辑客户
   - 删除客户（会同时删除关联配方）

## 技术栈
- Vue 3 + TypeScript
- TDesign Vue Next（UI组件库）
- Vue Router（路由管理）
- Pinia（状态管理）
- SCSS（样式预处理）

## 浏览器兼容性
所有动画和样式都使用标准CSS3属性，兼容所有主流浏览器：
- Chrome/Edge（最新版）
- Firefox（最新版）
- Safari（最新版）

## 使用说明
1. 登录后自动跳转到主页
2. 点击左侧导航栏可切换不同功能模块
3. 点击"新增"按钮可添加新数据
4. 表格中的"编辑"和"删除"按钮可对数据进行修改
5. 所有按钮都包含图标，点击时有明确的视觉反馈
