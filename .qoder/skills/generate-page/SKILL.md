# 生成页面

按 **project-ui-dynamic-theme** 规范开发页面，确保动态主题兼容。

## 开发要求

1. **严格还原**布局、交互、动画
2. 所有颜色使用 CSS 变量（`var(--color-primary)` 等），禁止写死色值
3. **组件化**、可复用、注释清晰
4. **响应式适配**（移动端/平板/桌面）
5. 包含**空状态**、**加载态**、**错误处理**

## 引用的技能和规则

- `project-ui-dynamic-theme` — 动态主题 UI 规范
- `vue-component` — Vue 组件规范
- `design-tokens` — 设计令牌规则
- `code-style` — 代码风格规范
